Projects

Automated Vegetable Patch System

I developed an innovative automated vegetable patch system that leverages sensor technology and IoT (Internet of Things) devices to optimize the growth conditions for plants. This project integrates sensors for monitoring crucial environmental variables such as humidity, solar light intensity, temperature, and soil moisture levels. By combining a web-based interface with ESP32 and Raspberry Pi 4 microcontrollers, this system offers real-time monitoring and control capabilities.
